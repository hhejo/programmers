## CODE 1

```javascript
function solution(n, a, b) {
  const { ceil } = Math;
  let ans = 0;
  while (a !== b) {
    a = ceil(a / 2);
    b = ceil(b / 2);
    ans += 1;
  }
  return ans;
}
```
