## Solution 1

Code

```javascript
function solution(food) {
  let ans = '';
  for (let [idx, cnt] of [...food.entries()].slice(1)) ans += idx.toString().repeat(Math.floor(cnt / 2));
  ans = `${ans}0${[...ans].reverse().join('')}`;
  return ans;
}
```
